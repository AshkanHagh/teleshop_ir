# Teleshop.ir

A Telegram Mini App for purchasing Telegram Premium and Stars using Zarinpal payment gateway. The platform includes a user-facing mini app and a realtime admin dashboard for managing products and monitoring transactions.

## Overview

Teleshop.ir enables users to buy Telegram Premium plans and Star packages directly within Telegram. Payments are processed securely through Zarinpal. The system features realtime price updates for TON-based products via scheduled jobs and WebSocket broadcasting.

### Key Features

- Telegram Mini App integration with init data verification
- Zarinpal payment processing with callback handling
- Realtime product price updates (TON to IRR conversion)
- User refresh/access token authentication
